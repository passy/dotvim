=========================
Passy's Vim Configuration
=========================

Installation
============

::

    git clone git@github.com:passy/dotvim.git ~/.vim
    ln -s ~/.vim/vimrc ~/.vimrc
    cd ~/.vim
    git submodule init && git submodule update

Submodule Update
================

::

    git submodule foreach git pull
